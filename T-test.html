Evaluation of the Stroop Phenomena

A Walkthrough of a Basic T-Test.

This page presents a walkthrough of a basic T-Test. It's not meant to be an explanation of what T-tests do, just an illustration of how one can use Python, Pandas and other libraries in data analysis and visualization. But just to give a completely oversimplified explanation of a T-Test, and statisitics in general, you're trying to determine if a pattern that you're seeing in data is "real" or not, and you're trying to extrapolate from a small sample of data to make statements about the entire population.

So the data that we'll use is on the stroop effect. The stroop effect involves the mind's ability to differntiate text from visual queues. The basic test is to time people naming a list of colors that are the same color as the word,termed Congruent, or colored differently from the word, termed Incongruent. All this means is, how long does it take you to read these two lists? DO NOT read the word: Say the color of the letters!

If you're at all like me, your brain gets tripped up on the Incongruent list and you want to read the word instead of say the color. You can learn more about the Stroop Effect here and the dataset used in this analysis can be found here.

So, now that you know what the Stroop Effect is, let's get started on the statistics. First, a few observations about the dataset, variable definitions, and so on:

  1. Question What is our independent variable? What is our dependent variable?

    Answer: The independent variable is Congruent vs. Incongruent classification. The dependent variable is the time it takes to name the color of inks in a given length list.

  2. Question What is an appropriate set of hypotheses for this task? What is the appropriate statistical test?

    Answer: Based on the times that I scored on the Stroop test, my hypotheses is that most people will find it more difficult to say the colors in the Incongurent list compared to the Congruent word list and therefore Incongruent times will be greater than Congruent Times. To put it more formally:

Terms, Variables & Constants Definition
Ha Hypothesis
H0 Null Hypothesis
$\mu_C$ Population Mean of Congruent Times
$\mu_I$ Population Mean of Incongruent Times
$\sigma_C$ Population Standard deviation of Congruent Times
$\sigma_I$ Population Standard deviation of Incongruent Times
$\hat{x}_C$ Sample Mean of Congruent Times
$\hat{x}_I$ Sample Mean of Incongruent Times
$S_C$ Sample Standard deviation of Congruent Times
$S_I$ Sample Standard deviation of Incongruent Times
$\alpha = 0.05$ Critical Value used to test significance
n=24 Sample size
$df = n-1$ degrees of freedom
$SE $ Standard Error: $SE = \frac{\sigma}{\sqrt{n}}$
  • Hypothesis Ha: Population mean time of Incongruent data will be significantly greater than Population Mean time of Congruent data. $$\mu_I > \mu_C $$
  • Null Hypothesis H0: There will be no significant difference between Population Mean times of Congruent and Incongruent datasets. $$\mu_C = \mu_I $$

Statistical Test:¶

  • Because the dataset consists of a sample of paired data values - congruent and incongruent times for each test subject, a One-Sided Dependent T-test is appropriate.

A T-test is appropriate because:

  1. The study is a within-subject design, and the dataset consists of dependent paired values
  2. The dataset consists of a relatively small sample (n=24), and there is no data available to describe the population. Because of these two issues, a T-test is appropriate instead of a Z-test.

  • Assumptions:

    1. The data is normally distributed, and therefore a T-test is valid.
    2. A critical value of 0.05 will be used to test for significance.
    3. A one sided test will be used based on anectdotal evidence from my own stroop test suggesting it is more time consuming to read Incongruent data than Congruent data.

Statistical Test¶

Okay, we've got a pretty good idea of what the data looks like and it's pretty obvious that Incongruent times are greater than Congruent times. the next step is to apply the T-Test.

Analysis:¶

Since $\hat{x}_C = 14.05$ and $\hat{x}_I = 22.02$ we can state that: $$ \hat{x}_I > \hat{x}_C$$
Next, we must determine if this is a significant difference. The statistical meaning of significant can be paraphrased as "does it pass the appropriate statistical test for the given critical value?"

As stated above we will use a one sided T-Test to determine significance.¶

In a real analysis one would use scipy's ttest function to quickly get the answer, but we'll go through the test step by step here: To do this, first calculate the Point Estimate of the Differences between Congruent and Incongruent Times: $\hat{x}_D = \hat{x}_I - \hat{x}_C = 22.02 - 14.05 = 7.97$

and the Sample Standard deviation of the differences as $$ S = \sqrt{\frac{\sum_{i=1}^n (t_{Ii} - t_{Ci})^2}{df}} $$

The Calculations are below:

T-Statistic:¶

  • The T-Statistic is calculated as the mean of the differences divided by the Standard Error:

$t = \frac{\hat{x}_D}{\frac{S}{\sqrt{n}}}$¶

In [12]:
t = x_D / (S/sqrt(n))
print't = %.3f / (%.3f / sqrt(%d)) = %.3f' %(x_D, S, n, t)

t = 7.965 / (4.865 / sqrt(24)) = 8.021

Observation: This is a pretty big t value!¶

Next, calculate T-Critical:¶

  • Remember that:
    $\alpha = .05$
    $df = 23$

I'll use scipy's probability density function to find $t_c$:¶

In [13]:
from scipy.stats import t
In [14]:
tAlpha = t.ppf(1-alpha, df) # use 1-alpha to return a positive t value.
round(tAlpha,3)
Out[14]:
1.714

For Comparison, The T-Value table confirms that for a one-sided significance test of $\alpha = 0.5$ with df=23 is $T_{critical} = 1.714$

Next, let's look at the probablity value of t:¶

  • a T value of 8.021 is off the scale on the T-table linked above, which means it's pretty close to zero. Lets use scipy to see how small the probablity is
In [15]:
#first double check that the probablity of tAlpha is .05
round(1 - t.cdf(tAlpha, df), 2)
Out[15]:
0.05
In [16]:
#Now look at the probablity value of t:
round(1-t.cdf(8.02, df),8)
Out[16]:
2e-08

Conclusion:¶

The t-value of 8.02 is significantly larger than the t-critical value of 1.714. $$ t > t_\alpha$$

In fact, the probablity value associated with t=8.02 is 2e-8, essentially zero. $$ p(t) \approx 0 $$ Therefore we can reject the null Hypothesis $H_0$ and accept the Alternative Hypothesis $H_A$ that:
$$\mu_I > \mu_C$$

This suggests that, for the given dataset and assumptions, congruent and incongruent times compose separate populations, and it takes significantly longer to read incongruent lists than congruent lists.

Now, Having gone through all the math, let's use scipy's ttest function to find the p-value:

* note: ttest returns a two-sided result which must be adjusted to one-sided.
In [18]:
from scipy.stats import ttest_rel
In [24]:
p = ttest_rel(StroopData.Congruent, StroopData.Incongruent)[1] * 2
print 'p = ',round(p, 8)

p =  8e-08

This returns the same probablity as the manual step-through above, and confirms that the differnece is significant. It's also A LOT faster than going through every step. Thanks for taking the time to read through this!
